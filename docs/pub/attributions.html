Attributions for the Daily Bulletin

Thank you all! Names are sorted by surname letters.

Project Founder

  • Albert Tan founded the project.

Editors

  • Albert Tan has been the editor until June 2024, and still provides suggestions.
  • Runxi Yu is the current maintainer of the project.

Designers

  • Albert Tan designed the original demo, wrote the CSS, and planned all the updates until June 2024.
  • Runxi Yu wrote the majority of the code for the modern iteration of the Daily Bulletin and occasionally updates the page design and the CSS.
  • An unknown person started the original primitive Daily Bulletin in or before 2022.

Tech Supports

  • Albert Tan wrote nearly all of the code and web pages for the project until June 2024.
  • Runxi Yu provided key inspirations, especially on eml file formats in October 2023; they took over the project in June 2024 and revamped nearly all the code.
  • ChatGPT taught us how to implement certain ideas and use certain libraries, especially BeautifulSoup.
